The following scripts are intende for use on Arch Linux and should be compatible with most Arch Linux derivative distributions, as they typically maintain compatibility with the core Arch Linux system. Arch Linux derivatives generally inherit package management, core utilities, and configurations from Arch Linux itself. 

Some notable Arch Linux derivatives include:
  -  Manjaro
  -  EndeavourOS
  -  ArcoLinux
  -  ArchBang
  -  Artix Linux (particularly the Arch-based version)
  -  Garuda Linux
  -  just to name a few.

I make this statement for new users that don't yet understand that Linux Distro's typically remain faithful to it's Parent Distro's package manager and repositories, appending their own customizations to the source. This is not true for all, but most distro derivatives.
